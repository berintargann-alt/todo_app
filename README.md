# Todo App

Bu proje ReactJS ve Tailwind CSS kullanılarak geliştirilmiş basit bir Todo uygulamasıdır.

## Proje Özellikleri

- Görev ekleme
- Görev listeleme
- Görev güncelleme
- Görev silme
- React component yapısı
- Tailwind CSS tasarımı
- Pages ve Components klasör yapısı

## Kullanılan Teknolojiler

- ReactJS
- Vite
- Tailwind CSS
- JavaScript

## Proje Yapısı

src/
 ├── components/
 │     └── TodoItem.jsx
 │
 ├── pages/
 │     └── HomePage.jsx
 │
 └── App.jsx

## Kurulum

```bash
npm install
npm run dev
```
